# Tool Associations

## Status

Concept note / future design direction.

This is not intended for immediate full implementation. The purpose is to capture the idea that NAM actions may be associated with the tools, applications, services, devices, or systems used to complete them.

This concept is related to expected artifacts, but separate.

Expected artifacts answer:

> What should exist when this work is done?

Tool associations answer:

> What do I use to get this work done?

## Core Idea

A NAM node may be associated with one or more tools.

Examples:

```text
Action:
  Send follow-up to Seafood Norway

Tool:
  Gmail
