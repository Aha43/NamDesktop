Blocked Actions

An action can be marked as blocked by one or more other actions. A blocked action is held out of your Next Actions and Backlog lists until every blocker is resolved — keeping your active lists clean and honest.

When you mark a blocking action as done, the blocked action automatically becomes available again and reappears in the appropriate list.

The Blocked panel

The Blocked entry in the sidebar shows all currently blocked actions, grouped by their blockers. Use it for a quick overview of what's being held up and why. Clicking a blocker opens its action dialog so you can work on it directly.

Adding a blocking relationship

Open the action dialog (double-click the action or press Enter). The Blocked by section shows a search field — type to find the action that must be done first, then select it. You can add as many blockers as needed.

Removing a blocking relationship

Open the action dialog and find the blocker in the Blocked by list. Click the remove button next to it to clear the relationship.

Cycle detection

The app prevents circular blocking chains — if A is blocked by B, you cannot then mark B as blocked by A. Any relationship that would create a cycle is rejected.
