Column view

Column view shows a project's structure as a board. When a project has sub-projects, open its workbench and click the columns icon in the breadcrumb bar.

  • Each column is one immediate sub-project.
  • Each card is an action belonging directly to that sub-project.
  • A leading Unsorted column appears when the project has actions of its own that don't yet live in a sub-project, so you can sort them into one.

It answers "which part of this project does this action belong to?" at a glance — something the stacked workbench sections can't show side by side.

Moving an action between columns

Select a card and click the move icon at the bottom of its column (or right-click the card), then pick the destination column. The action is reparented to that sub-project — Column view never invents a separate board state; moving a card simply changes which sub-project owns the action, exactly as if you had moved it in the workbench.

Click a column's title to drill into that sub-project. Use the columns icon again, or the readiness icon, to switch views; Column view is available whenever the project has at least one sub-project.
