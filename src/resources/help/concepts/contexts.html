Tag filter

The Tag filter panel lets you select one or more tags and see all matching actions across every project. Select @home and you see every action tagged with @home, wherever it lives.

Tags on a project are inherited by all actions inside it. Tagging a project @home automatically makes all its actions visible in the tag filter — without tagging each action individually.

The filter uses AND logic: selecting both @home and @computer shows only actions that match both tags.

You can save a tag combination as a Saved Filter so you can return to it quickly. For example, save @home + @computer as "Home office" for your home-working sessions.

Tip: tags like @phone, @computer, @errands work as GTD-style contexts — filter by location or tool to see what you can do right now.
