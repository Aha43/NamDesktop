Done

The Done list holds every action you have completed. Nothing is deleted automatically — finished work stays here so you can review it.

Done is most useful during your weekly review. Scan it to:

  • Confirm that a project is truly finished now that its last action is done
  • Spot recurring tasks you should turn into a standing commitment
  • Get a sense of how much you actually accomplished this week

Once an entry has served its purpose, delete it to keep the list readable. There is no rule about how long to keep items — a quick task can go immediately, while a milestone action might be worth keeping for a monthly or quarterly review.

Actions land here when you mark them Done from any panel — Inbox, Next Actions, Backlog, or the Project Workbench.
