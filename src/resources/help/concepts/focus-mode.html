Focus mode

Focus mode is a distraction-free execution mode that presents one action at a time from any lens view. The planning is already done — the lens is the plan. Focus mode is just a different way to consume it.

Click the deck icon on the Next Actions, Backlog, any Saved Filter panel, or a Project Workbench to enter focus mode. You see one card at a time: the action title, its description, and the project path. Press Space or click Done to mark it complete and advance to the next. Use ← → to browse without completing. Press Esc to exit back to the table.

In a Project Workbench the deck flips through that project's own open actions, so you can sit down and work a single project end to end.

Focus mode answers one question: what now?

Why one at a time?

When you are executing, you should not be navigating a bloated app. Showing one action removes the temptation to re-plan, re-prioritise, or pick something easier. The lens already decided what matters — focus mode just holds you to it.
