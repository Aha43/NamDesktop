Keyboard Shortcuts

On macOS use Cmd; on Windows/Linux use Ctrl.

Navigation

Cmd+1Go to Inbox
Cmd+2Go to Next Actions
Cmd+3Go to Backlog
Cmd+4Go to Projects
Cmd+5Go to Done
F1Open Help

Actions

Cmd+NNew item (context-sensitive: inbox item, project, or action)
Cmd+FSearch
Cmd+SSave / push workspace (shows nudge if sync not configured)
EnterOpen edit dialog for the selected row

View

Cmd+Shift+TShow / hide toolbar
Cmd+Shift+NShow / hide navigation pane
Cmd+Shift+ZZen mode (hide both toolbar and nav pane)
Cmd+/Open keyboard shortcuts reference dialog

AI / Monitoring mode

Cmd+Shift+MStart / stop monitoring mode
Cmd+Shift+SCheckpoint (review and accept AI changes)
