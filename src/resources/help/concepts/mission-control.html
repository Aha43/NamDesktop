Goal Board

A Goal Board is a high-level dashboard for tracking a set of related projects. It shows one card per project, coloured by how much of the work inside is done.

Green — most actions complete. Amber — in progress. Red — little or no progress yet.

Use Goal Boards for goals and life areas, not for day-to-day task management. Examples: Long-term goals, Home, Side projects.

To create a Goal Board: click New Goal Board… in the toolbar. Pick one or more tags — the preview shows which projects will appear immediately. Then give the board a name and click Create. Any project tagged with those tags will appear as a card on the dashboard.

Clicking a card opens the Project Workbench for that project.

Readiness view inside a project

The Workbench has a readiness view toggle (⊞) in the breadcrumb bar. When your project has sub-projects, clicking it switches to a heat-map view of those sub-projects — the same card layout as a Goal Board, but scoped to just that project's children. Click any card to drill in; use the back button to return to the readiness overview.
