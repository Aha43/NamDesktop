Mission Control

Mission Control is a high-level goal and area dashboard. It shows a heat-map of projects grouped by a tag — one card per project, coloured by how much of the work inside is done.

Green — most actions complete. Amber — in progress. Red — little or no progress yet.

Use Mission Control for goals and life areas, not for day-to-day task management. Examples: Long-term goals, Home, Side projects.

To create a Mission Control: click New Mission Control… in the toolbar, give it a name, and add one or more tags. Any project tagged with those tags will appear as a card on the dashboard.

Clicking a card opens the Project Workbench for that project, with the Mission Control name in the breadcrumb so you can navigate back to the dashboard.

MCR view inside a project

The Workbench has an MCR toggle (⊞) in the breadcrumb bar. When your project has sub-projects, clicking it switches to a heat-map view of those sub-projects — the same card layout as Mission Control, but scoped to just that project's children. Click any card to drill in; use the back button to return to the MCR overview.
