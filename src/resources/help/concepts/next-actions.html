Next Actions

A next action is the next physical step you can take to move something forward. "Organise the garage" is not a next action — it's a project. "Buy storage boxes" is a next action.

The test: could you sit down right now and do it? If yes, it's a next action. If it needs something else first, break it down further.

If you know GTD: this maps directly to your next-actions list — the committed, immediately doable steps.

In NamDesktop, Next Actions are all the tasks currently marked with Next status. They appear in the Next Actions view grouped and ordered. You can drag them up and down to set your own priority.

Each action shows its full project path so you always know where it belongs.

When you complete an action, right-click → Mark done. It moves to the Done list where you can review it later.

If an action isn't ready yet, move it to the Backlog so it doesn't clutter your active list.
