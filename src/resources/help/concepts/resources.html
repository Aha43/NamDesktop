Resources

Any action or project can carry one or more attached resources — links, file paths, email addresses, or freeform notes that belong with that item. Keep a reference URL next to the action that needs it, or attach a file path to a project so you always know where the related documents live.

Resource types

  • URI — a web link (e.g. https://example.com)
  • File — a local file path (e.g. /Users/you/Documents/brief.pdf)
  • Email — an email address to contact in relation to this item
  • Text — a freeform note that doesn't fit the other types

Each resource can have an optional description to remind you what it is.

Adding a resource

Open the action or project dialog (double-click the item, or click the pencil icon). Expand the Resources section, choose a type, enter a value and an optional description, then press Enter or click Add.

The paperclip indicator

A small paperclip icon appears in the list views — Inbox, Next Actions, Backlog, Done, Tag filter, and Projects — whenever a node has at least one resource attached. It's a quick signal that there's more context available without opening the item.

Removing a resource

Open the dialog, expand the Resources section, and click the remove button next to the entry you want to delete.
