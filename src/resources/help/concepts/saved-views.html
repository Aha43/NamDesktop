Saved Views

A Saved View is a named, persistent filter that shows all actions matching one or more tags. Create one for any combination you return to often — a project area, a location, a weekly focus — and it appears permanently in the left sidebar.

Creating a Saved View

Click the + button next to Saved Views in the sidebar. Enter a name and pick one or more tags. The view is added to the nav immediately.

The Next-only toggle

Each Saved View has a Next only option. When on, the view shows only actions with Next status — your committed, active work. When off, it shows all non-done actions matching the tags, including Backlog. Toggle it in the panel header to switch modes. The setting is saved per view.

Renaming and deleting

Use the Rename and Delete buttons in the panel header. Renaming keeps the tags and next-only setting intact.

How Saved Views differ from the Tag filter panel

The Tag filter panel is session-only — you pick tags, see results, and the selection resets when you navigate away. Saved Views are permanent: they live in the sidebar, keep their tag selection across sessions, and support the next-only toggle. Use the Tag filter for one-off lookups; use Saved Views for filters you reach for regularly.
