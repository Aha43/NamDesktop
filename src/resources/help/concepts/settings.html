Settings

Open Settings via File → Settings… in the menu bar. The dialog has three sections in the left sidebar.

Appearance

  • Theme — switch between Dark and Light mode. Applies immediately.
  • Dense mode — reduces row height across all list views. Useful on smaller screens.
  • Status column — adds a Status column to Next Actions and Backlog showing the status badge as text.
  • Start maximized — launches the app in a maximized window.

Workspace

  • Power user mode — restores the full inline toolbar in the Project Workbench: Rename, Edit description, and Delete buttons appear directly in the breadcrumb bar and sub-project headers.
  • Click to rename — when on, clicking an already-selected row title starts inline rename. Turn off to avoid accidental renames; all other rename paths (Rename button, right-click, double-click, Enter) still work.

Sync

  • Sync repo URL — URL of a private GitHub repository used for Git sync. Paste the repo URL here to enable Push and Pull in the toolbar and File menu.
