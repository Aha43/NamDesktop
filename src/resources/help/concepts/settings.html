Settings

Open Settings via File → Settings… in the menu bar. The dialog has three sections in the left sidebar.

Appearance

  • Theme — switch between Dark and Light mode. Applies immediately.
  • Icon color — Auto follows the theme (light icons in dark mode, dark in light); Light forces white icons and Dark forces black icons regardless of theme. Combine with Background brightness to dial in the contrast you want.
  • Background brightness — lightens the panel grays a few shades toward white, for a softer dark theme. Leave at 0 for the stock look. Applies immediately.
  • Dense mode — reduces row height across all list views. Useful on smaller screens.
  • Status column — adds a Status column to Next Actions and Backlog showing the status badge as text.
  • Start maximized — launches the app in a maximized window.

Workspace

  • Power user mode — restores the full inline toolbar in the Project Workbench: Rename, Edit description, and Delete buttons appear directly in the breadcrumb bar and sub-project headers.
  • Click to rename — when on, clicking an already-selected row title starts inline rename. Turn off to avoid accidental renames; all other rename paths (Rename button, right-click, double-click, Enter) still work.

Sync

Git sync

  • Sync repo URL — URL of a private GitHub repository used for Git sync. Paste the repo URL here to enable Push and Pull in the toolbar and File menu.

Cloud sync (Supabase) — see Cloud sync for the full story.

  • Enable — turns cloud sync on. The toolbar Push and Pull buttons then go to Supabase. When both Git and cloud sync are configured, cloud sync wins.
  • Supabase URL — where to sync to. Pre-filled for a local Supabase stack; paste your hosted project URL to sync to the cloud.
  • Publishable key — the project's public API key (sb_publishable_…). Pre-filled for the local stack.
  • Email / Password — your Supabase account credentials. The password field is masked; use Show to reveal it. Credentials are stored locally and only sent to the configured Supabase instance.
