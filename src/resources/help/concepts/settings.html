Settings

Open Settings via File → Settings… in the menu bar.

Theme

Switch between Dark and Light mode. The change applies immediately.

Dense mode

Reduces row height across all list views. Useful on smaller screens or when you prefer a more compact layout.

Power user mode

Restores the full inline toolbar in the Project Workbench: Rename, Edit description, and Delete buttons appear directly in the breadcrumb bar and sub-project section headers. Off by default to keep the interface clean for most users.

Show status column

Adds a Status column to the Next Actions and Backlog views showing the current status badge as text. Hidden by default since the coloured badge already conveys this.

Sync repo URL

The URL of a private GitHub repository used for Git sync. Paste the repo URL here to enable Push and Pull in the toolbar and File menu.
