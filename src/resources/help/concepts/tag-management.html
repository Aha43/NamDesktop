Tag Management

The Tag Management dialog gives you a central place to create, rename, and delete tags across your whole workspace.

Opening it

Go to File → Manage Tags… in the menu bar.

What you can do

  • New — create a tag and add it to the registry so it's available in all tag pickers
  • Rename — rename a tag; the change propagates instantly to every node carrying that tag
  • Delete — remove a tag from the registry and strip it from all nodes that had it

The table shows each tag and how many nodes currently use it, so you can spot orphaned or duplicate tags at a glance.

Tags can also be added and removed on individual actions and projects directly from their edit dialogs — Tag Management is for workspace-wide housekeeping.
