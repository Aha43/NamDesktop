Project Workbench

The Project Workbench is where you manage a single project in depth. Open it by clicking any project in the Projects list or by clicking a project path link in any action view.

The workbench shows:

  • A breadcrumb bar at the top — click any ancestor to navigate up
  • Sub-projects — each as a collapsible section with its own action list
  • This project — actions that belong directly to the current project

From the workbench you can:

  • Add actions and sub-projects
  • Quick-rename or edit descriptions inline
  • Delete sub-projects (with a warning showing how many items will be removed)
  • Navigate into any sub-project by clicking its name

MCR view — the name

MCR stands for Mission Control Room — the physical room at NASA where flight controllers sit at their consoles watching every sub-system of a mission at once. The MCR view inside the workbench does the same thing one level down: it gives you a console view of all your sub-projects so you can see the whole mission before diving into any single station.

If your project has sub-projects, the breadcrumb bar shows a dashboard icon (⊞). Click it to switch to MCR view — a heat-map of sub-projects showing progress at a glance. Click any card to drill into that sub-project; use Back to return to the MCR overview.
