Planning a Goal with a Goal Board

This tutorial walks through planning something big — like a home renovation, a career move, or preparing for retirement — using Projects and Goal Boards.

Step 1 — Create the top-level project

Go to Projects and click Add project. Name it something like Retirement planning. This is your goal container.

Step 2 — Break it into sub-projects

Open the project by clicking it — you land in the Project Workbench. Add sub-projects for the major milestones:

  • Financial baseline
  • Pension & savings review
  • Housing plan
  • Health & insurance

Each sub-project becomes its own planning area with actions inside it.

Step 3 — Tag your goal project

Select the top-level project and add a tag — for example @retirement. Tags on a project are inherited by all actions inside it, which lets you filter for everything related to this goal across the whole app.

Step 4 — Create a Goal Board

Click New Goal Board… in the toolbar (or File menu). Add the tag @retirement — the preview immediately shows your project. Name the board Long-term goals and click Create.

The board now shows a card for your retirement project with a progress indicator. As you complete actions inside, the card turns greener.

Step 5 — Use Readiness view for deep dives

Inside the Project Workbench, click the dashboard icon (⊞) in the breadcrumb bar. This switches to readiness view — a heat-map of all sub-projects. Click any card to drill into that sub-project. Use the back button to return to the readiness overview.

Step 6 — Work the actions

Open a sub-project, add concrete next actions inside it. Promote them to Next status when you're ready to act. They'll appear in your global Next Actions view with the full project path so you always know the context.

Tips

  • A Goal Board is great for reviewing progress across multiple related goals at once.
  • Keep actions small and physical — "Call pension provider" not "Sort out pension".
  • Use the Backlog to park actions that are real but not yet timely.
